# Changelog

## [1.3.0] - 2026-01-14

### Added
- 欢迎弹窗功能：在addClass页面添加了版本控制的欢迎弹窗
- 弹窗版本管理：支持通过更新版本号强制显示新弹窗内容
- 用户偏好存储：使用localStorage记住用户选择
- 自定义消息支持：可在HTML中轻松修改弹窗内容

## [1.2.1] - 2026-01-14

### Updated
- 添加显示课表的功能
- 将indexOf方法改回includes方法，使用ES2016的新特性
- 编译标准从ES2015升级到ES2016
- 更新README.md中的编译说明，将target参数从es2015改为es2016

## [1.2.0] - 2026-01-14

### Added
- 智能排课功能：从localStorage获取用户选择的课程组信息，生成最优排课方案
- 介意值配置：支持用户自定义四种介意值（时间冲突、校区冲突、早上第1节有课、晚上第11节有课）
- 排课算法：生成所有可能的选课方案并计算每种方案的总介意值
- 方案排序：按介意值排序，提供前5个最优方案
- 结果展示：显示每种方案的详细冲突情况和课程安排

### Features
- 课程分组逻辑：确保同一门课程（相同course_number）只能选择一个课堂
- 周次范围检查：支持普通周、单周、双周的课程安排
- 校区冲突检测：同一上午/下午超过两个课且校区不同时计算冲突
- 响应式设计：弹窗和按钮适配不同屏幕尺寸
- 状态管理：排课按钮仅在有选择课程组时可用

## [1.1.1] - 2026-01-13

### Updated
- 迷你课表样式改进：
  - 第一列（周日）和第六列（周五）右边添加加粗的虚线分割线
  - 第五行和第十行下面添加加粗的虚线分割线
  - 单元格高度从8px增加到12px，提供更好的视觉体验
- 课程详情弹窗功能增强：
  - 在弹窗中同时显示文字版的时间信息
  - 使用与搜索结果卡片相同的迷你课表样式

## [1.1.0] - 2026-01-13

### Added
- 前端HTML+CSS+TypeScript开发的课程组查询界面
- 课程组搜索功能，支持按课程编号、课程名称和课程组ID搜索
- 课程组选择功能，可将选择的课程组存储到localStorage
- 已选课程组实时展示功能
- 课程详情弹窗，展示完整课程信息
- 直观的课程表展示，包含13行7列的时间安排

### Features
- 响应式设计，适配不同屏幕尺寸
- 搜索结果分页展示
- 已选课程组可删除，自动更新localStorage
- 课程表第一列为周日，第5-6行和10-11行之间有明显间隙
- 课程时间区块用颜色填充，直观展示
- 课堂编号显示优化：当class_numbers超过10个时，显示为"很多"而不是全部列举

## [1.0.2] - 2026-01-12

### Added
- class_number_group功能：将课程按时间和校区信息分组
- 主程序main.py：一键执行完整数据处理流程
- 生成classNumberGroup.json文件：包含所有class_number_group的信息
- 生成classInfo_processed_with_group_id.json文件：添加了group_id的课程信息
- 生成class_info_data.js文件：包含课程信息数据、课程组数据和生成时间

### Features
- class_number_group_id格式优化：使用纯数字格式
- 分组逻辑完善：确保不同course_number的课程不会被分到同一个group
- 一键执行流程：自动按顺序执行所有处理步骤，显示清晰进度
- 文件检查机制：确保输入输出文件的存在性
- 跨平台兼容性：处理编码问题，确保在Windows环境下正常运行

## [1.0.1] - 2026-01-12

### Added
- 校区自动识别功能：根据地点自动判断校区编号
- 中文字段名转换为英文：统一使用英文字段名，方便后续开发
- 时间地点字段清理：删除其中的"_x000d_"特殊字符
- 课程编号拆分：将class_id拆分为course_number和class_number
- 更新了项目文档，添加了详细的数据格式说明和字段映射表

### Features
- 校区识别规则：
  - 0号校区：地点以"ARTS"开头
  - 1号校区：地点以"1"、"2"、"5"开头
  - 2号校区：地点以"3"开头
  - 3号校区：地点以"GT"开头
- 完整的字段名映射表，方便中英文名段对照
- 解析后的时间地点信息包含校区字段
- 课程编号和课堂编号拆分，便于课程管理
