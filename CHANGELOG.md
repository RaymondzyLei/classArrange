# Changelog

## [1.1.0] - 2026-01-13

### Added
- 前端HTML+CSS+TypeScript开发的课程组查询界面
- 课程组搜索功能，支持按课程编号、课程名称和课程组ID搜索
- 课程组选择功能，可将选择的课程组存储到localStorage
- 已选课程组实时展示功能
- 课程详情弹窗，展示完整课程信息
- 直观的课程表展示，包含13行7列的时间安排

### Features
- 响应式设计，适配不同屏幕尺寸
- 搜索结果分页展示
- 已选课程组可删除，自动更新localStorage
- 课程表第一列为周日，第5-6行和10-11行之间有明显间隙
- 课程时间区块用颜色填充，直观展示

## [1.0.2] - 2026-01-12

### Added
- class_number_group功能：将课程按时间和校区信息分组
- 主程序main.py：一键执行完整数据处理流程
- 生成classNumberGroup.json文件：包含所有class_number_group的信息
- 生成classInfo_processed_with_group_id.json文件：添加了group_id的课程信息
- 生成class_info_data.js文件：包含课程信息数据、课程组数据和生成时间

### Features
- class_number_group_id格式优化：使用纯数字格式
- 分组逻辑完善：确保不同course_number的课程不会被分到同一个group
- 一键执行流程：自动按顺序执行所有处理步骤，显示清晰进度
- 文件检查机制：确保输入输出文件的存在性
- 跨平台兼容性：处理编码问题，确保在Windows环境下正常运行

## [1.0.1] - 2026-01-12

### Added
- 校区自动识别功能：根据地点自动判断校区编号
- 中文字段名转换为英文：统一使用英文字段名，方便后续开发
- 时间地点字段清理：删除其中的"_x000d_"特殊字符
- 课程编号拆分：将class_id拆分为course_number和class_number
- 更新了项目文档，添加了详细的数据格式说明和字段映射表

### Features
- 校区识别规则：
  - 1号校区：地点以"1"、"2"、"5"开头
  - 2号校区：地点以"3"开头
  - 3号校区：地点以"GT"开头
- 完整的字段名映射表，方便中英文名段对照
- 解析后的时间地点信息包含校区字段
- 课程编号和课堂编号拆分，便于课程管理

## [1.0.0] - 2026-01-12

### Added
- 初始版本发布
- Excel转JSON功能实现
- 时间地点解析器功能实现，支持普通周、单周、双周格式
- 主处理程序实现，能够处理所有课程记录
- 项目文档（README.md）
- 依赖管理文件（requirements.txt）

### Features
- 支持从Excel文件读取课程信息并转换为JSON格式
- 支持解析复杂的时间地点格式，包括周次、地点、时间段
- 支持多个时间段的处理
- 程序健壮性强，能够处理None值和空字符串
- 详细的文档说明和使用示例
